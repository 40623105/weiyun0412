建立 Flask 網際應用程式

1.將 kmol_level2 解開到隨身碟中.

2.在 kmol_level2中附加 heroku_portable.7z,

3.複製%Disk%:\heroku\bin至set path1後面

4.利用學校的Gmail到 heroku 申請帳號.

5.在 Heroku cd2018 倉儲中 flask_ex1.py程式, Flask 改名為 app_run.py

6.新增3個檔案:

檔案Procfile,內容為:

web gunicorn app_run:app

      檔案 runtime.txt,內容為:

      python-3.6.2

      檔案requirements.txt,內容為:

      Flask==0.12.2

      gunicorn==19.7.1

      Jinja2==2.10

      MarkupSafe==1.0

      Werkzeug==0.14.1

7.在近端可攜程式命令列中, 輸入郵箱與密碼, 登入Heroku 帳號:

heroku login就能clone heroku 資料

8.接下來則必須在近端可攜程式命令列中, 打 heroku client 指令, 以及 git 指令, 準備要送到遠端的倉儲內容.

9.將 app_run.py, Procfile, runtime.txt 與 requirements.txt 等檔案放入 flask_heroku 目錄中,進入 flask_heroku 目錄, 以指令git init建立.git目錄

  1. 合併網頁,打指令git remote add github (合併網址).git

           git push github

11.以git 指令 add, commit 以及 push 提交推送到 Heroku的對應倉儲,以 git config 設定 user.name 以及 user.email, 才能提交推送,命令列必須位於 flask_heroku 目錄.

git add .
git commit -m "自訂名稱"
git push
